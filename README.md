
# BalkanaTry Web Application

## About BalkanaTry project
Web Application for routes in Balkana, Tryavna area about:

```
Mountain running
Mountain cycling
Mountain hiking
Road cycling
Adventures
Sharing information
```

## Used Technologies
```
Framework - Next.js
Database - Supabase
Styling - Tailwindcss
```
